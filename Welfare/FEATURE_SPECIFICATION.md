# 복지24 프로젝트 기능 명세서

## 프로젝트 정보
- **프로젝트명**: 복지24 (Welfare24)
- **개발 기간**: 2024.09 - 2024.10
- **개발 인원**: 1명
- **기술 스택**: Spring 5.0.7, MyBatis 3.5.9, MySQL 8.3.0, JSP, JavaScript

---

## 기능 명세

| 구분 | 요구사항 | 요구사항 내용 |
|---|---|---|
| **회원가입** | 아이디 | 영문, 숫자 조합 4~50자 |
| | 비밀번호 | 영문, 숫자, 특수문자 조합 8~255자 |
| | 이름 | 한글 2~100자 |
| | 이메일 | 이메일 형식, 중복 확인 필수 |
| | 전화번호 | 하이픈 포함 20자 이내 (예: 010-1234-5678) |
| | 생년월일 | DATE 형식 (YYYY-MM-DD) |
| | 권한 | USER / ADMIN (기본값: USER) |
| | 가입일 | 자동 생성 (CURRENT_TIMESTAMP) |
| | 회원가입 버튼 | submit |
| | 중복 확인 | 아이디, 이메일 중복 체크 AJAX |
| **로그인** | 일반 로그인 | 아이디, 비밀번호 일치 확인 후 메인 페이지 이동 |
| | 세션 관리 | JSP session에 id, name 저장 |
| | 로그인 유지 | 체크박스, 쿠키 저장 (7일) |
| | 아이디 찾기 | 이름, 이메일로 아이디 찾기 |
| | 비밀번호 찾기 | 이름, 이메일로 임시 비밀번호 발급 |
| | 로그인 버튼 | submit |
| | 로그아웃 | 세션 무효화 후 메인 페이지 이동 |
| **메인페이지** | 복지 진단 배너 | 메인 배너 슬라이드 (이미지 3개 자동 전환) |
| | 인기 복지 서비스 | 조회수 기준 TOP 10 슬라이더 |
| | 기부하기 섹션 | 9개 카테고리 카드 레이아웃 |
| | 봉사활동 섹션 | 모집 중인 활동 3개 미리보기 |
| | 스크롤 애니메이션 | Intersection Observer API 활용 fade-in |
| | 푸터 정보 | 사업자 정보, 연락처, 이용약관 링크 |
| **복지 진단** | 진단 유형 선택 | 기본 진단 / 상세 진단 (2가지) |
| | 생년월일 | DATE 형식, 나이 계산용 |
| | 성별 | radio 버튼 (남성/여성) |
| | 가구원 수 | number input (1~20) |
| | 소득 수준 | select 드롭다운 (중위소득 50%, 75%, 100% 등) |
| | 결혼 상태 | radio 버튼 (미혼/기혼/이혼/사별) |
| | 자녀 수 | number input (0~10) |
| | 취업 상태 | select 드롭다운 (취업/미취업/학생/주부 등) |
| | 지역 선택 (시도) | select 드롭다운 (서울, 경기, 부산 등) |
| | 지역 선택 (시군구) | select 드롭다운 (시도 선택에 따라 동적 변경) |
| | 임신 여부 | checkbox (상세 진단만) |
| | 장애 여부 | checkbox (상세 진단만) |
| | 다문화 가정 | checkbox (상세 진단만) |
| | 보훈 대상 | checkbox (상세 진단만) |
| | 한부모 가정 | checkbox (상세 진단만) |
| | 진단 버튼 | submit |
| | 로그인 여부 확인 | 상세 진단은 로그인 필요, 기본 진단은 비로그인 가능 |
| | 결과 저장 | 로그인 사용자만 진단 결과 자동 저장 |
| **복지 매칭 결과** | 매칭 알고리즘 | 점수 기반 (생애주기 30점, 가구유형 40점, 지역 10점 등) |
| | 최소 점수 | 10점 이상 매칭된 서비스만 표시 |
| | 서비스 목록 | 카드 레이아웃, 서비스명, 소관부처, 지원유형 표시 |
| | 신청 방법 | 온라인/방문/전화 신청 방법 안내 |
| | 관심 서비스 등록 | 북마크 버튼 (로그인 필요) |
| | 재진단 버튼 | 진단 페이지로 이동 |
| **외부 API 연동** | 중앙부처 복지 API | 공공데이터포털 NationalWelfareInformationsV001 |
| | 지자체 복지 API | 공공데이터포털 LocalGovernmentWelfareInformations |
| | API KEY | 서비스에 하드코딩 (환경변수 권장) |
| | 응답 형식 | XML → JSON 파싱 |
| | 에러 처리 | API 호출 실패 시 기본 메시지 표시 |
| **관심 복지 서비스** | 북마크 추가 | 로그인 후 복지 서비스 관심 등록 |
| | 중복 방지 | 복합 기본 키 (user_id, service_id) |
| | 북마크 목록 | 마이페이지에서 조회, 최신순 정렬 |
| | 북마크 삭제 | 개별 삭제 버튼 |
| **기부하기** | 기부 유형 | radio 버튼 (정기 후원 / 일시 후원) |
| | 기부 카테고리 | select 드롭다운 (위기가정, 의료비, 화재피해 등 9개) |
| | 후원 패키지 | radio 버튼 (3만원, 5만원, 10만원, 직접 입력) |
| | 기부 금액 | number input, DECIMAL(15,2) |
| | 후원자명 | text input (필수) |
| | 후원자 이메일 | email input, 형식 검증 (필수) |
| | 후원자 전화번호 | tel input (선택) |
| | 후원 메시지 | textarea (선택) |
| | 결제 상태 | pending / completed / failed (자동 처리) |
| | 비회원 기부 | 로그인 없이 기부 가능 |
| | 기부 버튼 | submit |
| **기부 후기** | 후기 작성 | 로그인/비로그인 모두 가능 |
| | 작성자명 | text input (필수) |
| | 별점 | 1~5점 (star rating UI) |
| | 후기 내용 | textarea 500자 이내 (필수) |
| | 익명 여부 | checkbox (익명으로 작성) |
| | 도움됨 카운트 | 후기 추천 버튼 (좋아요) |
| | 후기 목록 | 최신순, 별점순 정렬 select |
| | 후기 수정/삭제 | 작성자 본인만 가능 |
| **봉사 활동** | 활동 목록 | 모집 중인 봉사 활동 리스트 |
| | 활동명 | VARCHAR(200) |
| | 활동 설명 | TEXT 형식 |
| | 봉사 장소 | VARCHAR(200) |
| | 봉사 날짜 | DATE 형식 |
| | 봉사 시간 | 시작 시간 ~ 종료 시간 (TIME) |
| | 모집 인원 | 현재 인원 / 최대 인원 표시 |
| | 모집 상태 | recruiting / closed / completed |
| | 신청하기 버튼 | 모집 중일 때만 활성화 |
| | 상세보기 | 모달 팝업 또는 상세 페이지 이동 |
| **봉사 신청** | 신청 폼 UI | 3단계 위자드 (카테고리 선택 → 정보 입력 → 확인) |
| | 활동 선택 | 봉사 활동 ID 자동 전달 |
| | 신청자명 | text input (필수) |
| | 전화번호 | tel input (필수) |
| | 이메일 | email input (선택) |
| | 주소 | Daum 우편번호 API 연동 |
| | 봉사 경험 | radio 버튼 (없음/1년미만/1-3년/3년이상) |
| | 봉사 분야 | select 드롭다운 (노인, 아동, 환경 등) |
| | 봉사 날짜 | DATE 선택 (시작일, 종료일) |
| | 봉사 시간대 | select 드롭다운 (오전/오후/종일) |
| | 신청 상태 | applied (신청 시 자동 설정) |
| | 비회원 신청 | 로그인 없이 신청 가능 |
| | 신청 완료 버튼 | submit, 정원 초과 시 경고 메시지 |
| **봉사 후기** | 후기 작성 권한 | 봉사 상태가 'completed'일 때만 가능 |
| | 작성자명 | text input (필수) |
| | 후기 제목 | text input 200자 이내 (필수) |
| | 후기 내용 | textarea 1000자 이내 (필수) |
| | 별점 | 1~5점 (star rating UI) |
| | 비회원 후기 | 로그인 없이 작성 가능 |
| | 후기 목록 | 활동별 후기 조회 |
| | 후기 삭제 | 작성자 본인 또는 관리자만 가능 |
| **복지 지도** | 지도 API | Kakao Maps JavaScript API |
| | 지도 표시 | 초기 위치: 서울시청 |
| | 현재 위치 | Geolocation API로 사용자 위치 마커 |
| | 복지 시설 검색 | 키워드 입력 (복지관, 주민센터 등) |
| | 마커 표시 | 검색 결과 지도에 마커 표시 |
| | 정보 창 | 마커 클릭 시 시설명, 주소, 전화번호 표시 |
| | 길찾기 | Kakao 길찾기 API 연동 |
| **공지사항** | 공지 목록 | 전체 공지사항 페이징 처리 (10개씩) |
| | 상단 고정 | is_pinned=TRUE인 공지 최상단 표시 |
| | 제목 | VARCHAR(200) |
| | 내용 | TEXT 형식, HTML 에디터 (CKEditor 권장) |
| | 작성자 | 관리자 이름 표시 |
| | 작성일 | TIMESTAMP 형식, YYYY-MM-DD HH:mm 표시 |
| | 조회수 | 공지 조회 시 +1 증가 |
| | 작성 권한 | 관리자(ADMIN)만 가능 |
| | 수정/삭제 | 관리자(ADMIN)만 가능 |
| **FAQ** | 카테고리 | 복지혜택, 서비스이용, 계정관리, 기타 (4개) |
| | 카테고리 탭 | 탭 UI로 카테고리별 필터링 |
| | 질문 | VARCHAR(500) |
| | 답변 | TEXT 형식 |
| | 정렬 순서 | order_num 기준 오름차순 |
| | 활성화 여부 | is_active=TRUE인 FAQ만 노출 |
| | 아코디언 UI | 질문 클릭 시 답변 토글 |
| | 관리 권한 | 관리자(ADMIN)만 작성/수정/삭제 |
| **마이페이지** | 회원 정보 조회 | 로그인 필요, 본인 정보만 조회 |
| | 회원 정보 수정 | 이름, 전화번호, 이메일 수정 가능 |
| | 비밀번호 변경 | 현재 비밀번호 확인 후 새 비밀번호 설정 |
| | 나의 진단 내역 | 복지 진단 이력 테이블 (날짜, 매칭 서비스 수) |
| | 나의 기부 내역 | 기부 내역 테이블 (날짜, 금액, 분야, 상태) |
| | 나의 봉사 신청 | 봉사 신청 내역 및 상태 조회 |
| | 관심 복지 서비스 | 북마크한 복지 서비스 목록 |
| | 회원 탈퇴 | 비밀번호 확인 후 탈퇴, 관련 데이터 SET NULL |
| **네비게이션** | 상단 메뉴 | 복지 진단, 복지 지도, 기부하기, 봉사활동, 공지사항, FAQ |
| | 로그인 상태 표시 | 로그인 시: 닉네임, 로그아웃 / 비로그인 시: 로그인, 회원가입 |
| | 로고 클릭 | 메인 페이지로 이동 |
| | 반응형 메뉴 | 모바일에서 햄버거 메뉴 |
| **검색** | 통합 검색 | 공지사항, FAQ 제목+내용 검색 |
| | 검색 키워드 | text input, 2자 이상 입력 시 검색 |
| | 검색 결과 | 검색어 하이라이트 표시 |
| | 검색 결과 없음 | "검색 결과가 없습니다" 메시지 |
| **주소 검색** | 우편번호 API | Daum 우편번호 서비스 |
| | 팝업 레이어 | 새 창 또는 iframe 팝업 |
| | 주소 형식 | 도로명 주소 / 지번 주소 선택 |
| | 상세 주소 | text input 사용자 직접 입력 |
| | 주소 자동 입력 | API 콜백으로 주소 input에 자동 입력 |
